Taggest.com
==========

Requirements
============
* Node.js ~0.10.24
* Express ~4.6.1
* Jade ~1.4.2

Usage
=====
* Install Node.js & npm
* Use ```$ npm install .``` in project dir to install dependencies
* Use ```$ node ./index.js``` in project dir to start server
* Use ```$ guard``` in project dir to compile SCSS and start Livereload (optional)
* Go to http://localhost:5000/
